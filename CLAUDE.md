# fluid-stdlib

Kotlin Multiplatform utility library with potentially useful standard library additions (JVM + JS).

## Build

Uses `io.fluidsonic.gradle` plugin. Build and test: `./gradlew build`.

## Conventions

- Tab indentation in all Kotlin and Gradle files.
- Release tags have no `v` prefix (e.g. `0.14.0`, not `v0.14.0`).
- Source directories: `sources/` and `tests/` (not `src/`), configured by fluid-gradle plugin.
